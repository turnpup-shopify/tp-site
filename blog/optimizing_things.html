How to CRO any page?

Understand the Bigger Picture

  1. Articulate the vision.
  2. What role does this page play in getting there?
  3. What metric or set of metrics best describes if we are doing that successfully?

Always Applicable

  1. Copy that gets directly to the point.
  2. Cut noise.
  3. Legibility.
  4. Buttons should be easily clickable.
