Rhino 3D Shoe Modeling

The goal of this project was to learn how to use Rhino by turning one of my sketches into a 3D model using rhino.
I started with an import of a last and built the midsole around that. Working upwards I included a thin shell to help encase the upper of the shoe.

Solidworks 3D Shoe Modeling

The goal of this was to experiment with solidworks 3D modeling to see if I could crete a model for one of my sketches. The idea was to create a modern shoe that would be durable and lightweight.
